## ğŸ§© Komponen yang Tersedia

### 1. Header / Navigation
- Logo perusahaan di sisi kiri.  
- Menu navigasi responsif dengan efek transisi halus.  
- Tampil sebagai *hamburger menu* di perangkat mobile tanpa JavaScript.

### 2. Hero Section
- Headline kuat dan inspiratif yang merepresentasikan perusahaan teknologi masa depan.  
- Subtitle penjelas singkat tentang layanan atau visi perusahaan.  
- CTA (*Call To Action*) button dengan efek hover halus dan transisi smooth.

### 3. Features Section
- Menampilkan **3 fitur utama** perusahaan.  
- Setiap fitur memiliki **ikon glow biru neon** dan deskripsi singkat.  
- Menggunakan grid responsif untuk menyesuaikan layout di berbagai ukuran layar.

### 4. About Section
- Informasi singkat mengenai perusahaan, visi, dan nilai-nilai inti.  
- Desain clean dengan efek *fade-in* transisi lembut saat di-scroll.

### 5. Contact Form
- Form kontak dengan input bergaya modern dan fokus pada keterbacaan.  
- Menggunakan warna kontras lembut dengan efek fokus (*focus ring*) Tailwind.  
- Dapat digunakan untuk simulasi pengiriman pesan pelanggan.

### 6. Footer
- Informasi hak cipta perusahaan.  
- Link ke media sosial seperti **Facebook**, **GitHub**, **Instagram**, dan **WhatsApp**.  
- Setiap ikon memiliki efek **glow biru neon** saat di-hover.
